📡 NVD (National Vulnerability Database)

CVSS scores and full CVE descriptions from NIST's NVD API v2. Enriches kernel CVE findings with severity and impact data.

🔗 nvd.nist.gov

⚠️ CISA KEV Catalog

CISA's Known Exploited Vulnerabilities catalog — CVEs actively exploited in the wild. Findings matching KEV entries are flagged as highest priority.

🔗 cisa.gov/kev

📊 EPSS Scores

Exploit Prediction Scoring System — probability a CVE will be exploited in the next 30 days. Used to boost priority scores for high-exploitation-probability findings.

🔗 first.org/epss

🛡️ CVE.org (MITRE CVE Program)

Official MITRE CVE database (CVE JSON 5.0). Provides CWE weakness classification, affected vendor/product list, and authoritative reference links for each CVE.

🔗 cve.org  ·  API docs
